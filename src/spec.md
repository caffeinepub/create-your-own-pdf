# Specification

## Summary
**Goal:** Add Tamil language support to the PDF translator and fix download functionality across all PDF tools.

**Planned changes:**
- Add Tamil as both source and target language option in PDF translator dropdowns
- Fix download functionality in Image to PDF converter to generate and download combined PDF
- Fix download functionality in PDF to Image converter to generate and download individual images or zip archive
- Fix download functionality in PDF Splitter to generate and download split PDF with selected pages
- Fix download functionality in PDF Merge tool to generate and download single combined PDF
- Fix download functionality in Document to PDF converter to generate and download PDF from DOCX/TXT/RTF
- Fix download functionality in PDF to Document converter to generate and download DOCX/TXT files
- Fix download functionality in PDF Unlock tool to generate and download unlocked PDF
- Fix download functionality in PDF Editor to generate and download edited PDF with all annotations
- Verify and ensure PDF Translator properly extracts text, translates via LibreTranslate API, and generates downloadable translated PDF

**User-visible outcome:** Users can select Tamil as a translation language and successfully download processed files from all PDF tools without errors.
